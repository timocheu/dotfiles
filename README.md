# Setup (LINUX) ⚡
### Stowing Instruction
1. Install a stower, ex. 'stow', 'xstow' or 'rstow'
2. Setup Ex. `stow -t ~/.config/nvim dotfiles/nvim`
3. If added new file do a restow
